1 Google Sheets bazasini yaratish

  1. Google Sheets ga kiring
  2. Yangi jadval yarating va nomini Oila Market DB deb qo'ying
  3. URL dan Spreadsheet ID ni oling: 
    https://docs.google.com/spreadsheets/d/SPREADSHEET_ID_SHU_YERDA/edit
  4. ID ni nusxalab oling - keyinchalik kerak bo'ladi
💡 Har bir jadval (sheet) avtomatik yaratiladi - qo'lda qo'shish shart emas!

2 Google Apps Script sozlash

  1. Google Apps Script ga kiring
  2. "Yangi loyiha" (New Project) bosing
  3. Loyiha nomini Oila Market API deb qo'ying
  4. Code.gs faylini oching va barcha kodni o'chirib, bizning Code.gs faylidagi kodni to'liq joylashtiring
  5. Yuqoridagi qatorda SPREADSHEET_ID ni o'z ID ingiz bilan almashtiring: 
    const SPREADSHEET_ID = 'sizning_spreadsheet_id';
  6. IMGBB_API_KEY ni ham o'z kalitingiz bilan almashtiring (3-qadamda olinadi)
  7. Deploy → New deployment bosing
  8. Turini "Web app" qilib tanlang
  9. Sozlamalar:
    • Execute as: Me
    • Who has access: Anyone
  10. Deploy bosing va berilgan URL ni nusxalang: 
    https://script.google.com/macros/s/AKfycby.../exec
⚠️ Muhim: "Anyone" ruxsatini bering, aks holda API ishlamaydi!

3 ImgBB API kalitini olish

  1. imgbb.com ga kiring
  2. Ro'yxatdan o'ting yoki kiring
  3. API sahifasiga o'ting
  4. "Add API Key" bosing
  5. API kalitni nusxalang
  6. Bu kalitni Code.gs dagi IMGBB_API_KEY ga joylashtiring
💡 ImgBB - bepul rasm hosting servisi. Mahsulot rasmlari, banner va profil rasmlarini saqlash uchun ishlatiladi.

4 Kodlarni sozlash

4.1 - indexuser.html

Fayl boshidagi API o'zgaruvchisini o'z Apps Script URL ingiz bilan almashtiring:

const API = 'https://script.google.com/macros/s/SIZNING_URL/exec';

4.2 - indexadmin.html

Xuddi shunday API o'zgaruvchisini yangilang:

const API = 'https://script.google.com/macros/s/SIZNING_URL/exec';

4.3 - Code.gs

Quyidagi o'zgaruvchilarni yangilang:

const SPREADSHEET_ID = 'sizning_spreadsheet_id';
const IMGBB_API_KEY = 'sizning_imgbb_api_kalit';

5 GitHub Pages ga joylashtirish

  1. GitHub da yangi repository yarating (masalan: Oila-Market)
  2. Quyidagi fayllarni yuklang:
    Oila-Market/
    ├── index.html         (indexuser.html nusxasi)
    ├── indexadmin.html  (admin panel)
    └── Code.gs          (faqat reference uchun)
  3. Muhim: indexuser.html nomini index.html ga o'zgartiring!
  4. Repository Settings → Pages ga o'ting
  5. Source ni "Deploy from a branch" qilib tanlang
  6. Branch: main, Folder: / (root)
  7. Save bosing
  8. Bir necha daqiqadan so'ng saytingiz tayyor: 
    https://username.github.io/Oila-Market/          (User App)
https://username.github.io/Oila-Market/indexadmin.html  (Admin App)
✅ GitHub Pages - bepul va cheksiz hosting!

Git buyruqlari (terminal orqali):

git init
git add index.html indexadmin.html
git commit -m "Initial commit - Oila Market"
git branch -M main
git remote add origin https://github.com/USERNAME/Oila-Market.git
git push -u origin main

6 Telegram Bot yaratish (ixtiyoriy)

  1. Telegram da @BotFather ga murojaat qiling
  2. /newbot buyrug'ini yuboring
  3. Bot nomi va username kiriting
  4. BotFather sizga API token beradi
  5. Bot sozlamalarida Web App qo'shing: 
    /setmenubutton → Web App URL ni kiriting
  6. User app URL: https://username.github.io/Oila-Market/
  7. Admin bot uchun alohida bot yarating va URL: .../indexadmin.html
💡 Telegram bot orqali foydalanuvchilar to'g'ridan-to'g'ri ilovaga kirishlari mumkin!
